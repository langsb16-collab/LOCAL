# 지자체형 사회적기업 일자리 매칭 플랫폼

## 프로젝트 개요
- **이름**: 일자리 매칭 플랫폼 (Job Matching Platform)
- **목표**: 노령층, 여성, 장애인을 위한 맞춤형 일자리 매칭 서비스 제공
- **주요 대상**: 노령층, 여성, 장애인 구직자 및 지자체 고용담당자

## 주요 기능

### ✅ 완료된 기능
1. **다국어 지원 (7개 언어)** 🌍
   - 한국어, 영어, 중국어, 일본어, 베트남어, 몽골어, 러시아어
   - 실시간 언어 전환 기능
   - 모든 UI 텍스트 다국어 지원

2. **모바일 최적화 반응형 디자인** 📱
   - 휴대폰 화면에 최적화된 UI/UX
   - 컴팩트한 섹션 크기 및 글자 크기
   - 최소화된 빈 공간으로 효율적인 화면 활용
   - 부드러운 스크롤 경험
   - Sticky 헤더로 편리한 네비게이션
   - 터치 친화적인 버튼 크기

3. **일자리 검색 및 필터링**
   - 대상 유형별 검색 (노령층/여성/장애인/전체)
   - 지자체별 필터링
   - 업종별 필터링
   - 키워드 검색 기능
   - 실시간 검색 결과 업데이트

4. **통계 대시보드**
   - 전체 일자리 수
   - 전체 지원자 수
   - 전체 사용자 수
   - 대상별 일자리 분포
   - 지자체별 일자리 분포
   - 모바일에서 보기 쉬운 카드 레이아웃

5. **일자리 상세 정보**
   - 회사명, 근무지, 급여
   - 근무 시간, 고용 형태
   - 필요 기술 및 복리후생
   - 조회수 추적
   - 간결한 카드 형식으로 정보 제공

6. **데이터베이스 설계**
   - 지자체 정보 관리
   - 사용자 프로필 관리
   - 일자리 카테고리 관리
   - 일자리 공고 관리
   - 지원 현황 관리

## 데이터 구조

### 주요 테이블
1. **municipalities** - 지자체 정보
   - 7개 언어 지원 (이름)
   - 지역, 연락처 정보

2. **users** - 사용자 정보
   - 이메일, 이름, 연락처
   - 사용자 유형 (노령층/여성/장애인)
   - 선호 언어, 지자체 연결

3. **job_categories** - 일자리 카테고리
   - 7개 언어 지원
   - 아이콘 정보

4. **jobs** - 일자리 공고
   - 7개 언어 지원 (제목, 설명)
   - 급여, 근무 시간, 고용 형태
   - 대상 유형, 마감일
   - 조회수, 상태 관리

5. **applications** - 지원 현황
   - 사용자-일자리 매칭
   - 지원 상태 추적
   - 자기소개서

### 저장 서비스
- **Cloudflare D1** - SQLite 기반 글로벌 분산 데이터베이스
  - 로컬 개발: `.wrangler/state/v3/d1` (SQLite)
  - 프로덕션: Cloudflare D1 데이터베이스

   - 부산제조
   - 월 210-260만원

4. **학습지도 보조교사** (여성 대상, 시간제)
   - 인천교육지원센터
   - 월 230-300만원

5. **요양보호사** (전체 대상, 정규직)
   - 서울요양원
   - 월 250-320만원

- **Hono** - 경량 웹 프레임워크

### 프론트엔드
- **TailwindCSS** - 스타일링 (CDN)
- **Font Awesome** - 아이콘 (CDN)
- **Axios** - HTTP 클라이언트 (CDN)
- **Vanilla JavaScript** - UI 인터랙션

### 개발 도구
- **Vite** - 빌드 도구
- **Wrangler** - Cloudflare CLI
- **PM2** - 프로세스 관리 (개발 환경)

## 로컬 개발

### 사전 요구사항
- Node.js 18+
- npm 또는 pnpm

### 설치 및 실행
```bash
# 의존성 설치
npm install

# 프로젝트 빌드
npm run build

# D1 마이그레이션 (로컬)
npm run db:migrate:local

# 시드 데이터 추가
npm run db:seed

# 개발 서버 시작 (PM2 사용)
pm2 start ecosystem.config.cjs

# 또는 직접 실행
npm run dev:sandbox

# 로그 확인
pm2 logs job-matching --nostream
```

### 유용한 명령어
```bash
# 포트 정리
npm run clean-port

# 데이터베이스 초기화
npm run db:reset

# 빌드
npm run build

# 타입 생성
npm run cf-typegen
```
